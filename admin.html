Quick Setup Guide

1. Create Cloudflare R2 Bucket

  1. Go to Cloudflare Dashboard
  2. Navigate to R2 Object Storage
  3. Click "Create Bucket"
  4. Name it (e.g., "portfolio-media")

2. Generate API Token

  1. Go to My Profile ‚Üí API Tokens
  2. Create Token ‚Üí Use "Edit Cloudflare Workers" template
  3. Add R2 permissions: Read and Write
  4. Copy the token

3. Enable Public Access (Optional)

  1. In your R2 bucket settings
  2. Enable "Public Access"
  3. Copy the public URL
